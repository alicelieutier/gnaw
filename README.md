# Gnaw

A handy reminderbot for students and coaches alike.

## Quickstart

Check out `env.sh.example`, fill out your own, and stick it in `env.sh`

To run the tests

```
npm test
```

To see the usage

```
./run_with_env node run.js
```

## Server

It's on heroku, scheduled to run at 9.30am every day.
